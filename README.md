# 幻想少女的可爱日常（LovelyDaily）
## 介绍（Introduction）

这是一本描写二分之一少女之间日常的幻想小说。

部分内容由作者的经历改编。书中的很多人物你都能找到生活中的原型，活跃着的他们会影响本书日后的发展。试着从这浅陋的笔法中，窥见他们的经历？
